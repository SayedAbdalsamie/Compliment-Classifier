# CITIOAI

This folder contains the main services and files for the CITIOAI project.

## Structure
- `chatbot/`: Chatbot service and helpers
- `classifer/`: Classifier service, models, and helpers
- `moderator/`: Moderator service and helpers
- `app.py`: Main application entry point
- `requirements.txt`: Python dependencies

Sensitive files and models are excluded from version control for security.
